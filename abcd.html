Пусть исполняемая программа хранится в стороке
str:string[100];
переменные a,b,c,d хранятся в массиве
arr:array[0..3];

Инициализируем переменные
str:=' a=2 b=3 ';
Для обработки символов a,b,c,d в тексте программы (сторке srt[]), используется оператор Ord, который преобразует
букву в её числовой код из ascii-таблицы.
ascii-коды символов, соответствующих буквам a,b,c,d лежат в диапазоне от 97 до 100 

  var
  test_char:char;
  begin
    test_char:='a';
    writeln(Ord(test_char));
  end.
код программы здесь https://github.com/demsp/pas/blob/master/abcd/Ord.pas
Для посимвольного перебора строки используется цикл while.
Программа по ссылке выводит на экран символы a,b
https://github.com/demsp/pas/blob/master/abcd/workpiece-billet.pas
Пускай ascii-коды символов соответствующих переменных, получаемые в результате посимвольного перебора строки str[]
загружаются в аккумулятор var_acc
var_acc:=Ord(str[i]);

ascii-коды символов, соответствующих числам, лежат в диапазоне от 48 до 58
Пускай цифра, полученная в результате посимвольного перебора строки, загружается в переменную, ascii-код которой
хранится в переменной var_acc
if Ord(str[i])>=48 then if Ord(str[i])<=58 then arr[var_acc-97]:=Ord(str[i])-48;
https://github.com/demsp/pas/blob/master/abcd/load_digit_to_arr.pas

Для передачи данных из одной переменной в другую добавим оператор точка-с-запятой
Флаг сбрасывается при обработке данного символа
if str[i]=';' then flag:=false
https://github.com/demsp/pas/blob/master/abcd/digit_letter_acc_arr.pas

Для разбора арифметического выражения, содержащего + добавим аккумулятор acc для хранения полученной суммы Если при разборе строки попалось числовое значение, то оно прибавляется к аккумулятору
acc:=acc+(Ord(str[i])-48);
Если встретилось имя переменой, то к acc прибавляется значение, содержащиеся в этой переменной
acc:=acc+arr[Ord(str[i])-97];
https://github.com/demsp/pas/blob/master/abcd/plus_expression.pas

Пускай цикл обрабатывается внутри квадратных скобок
Пускай переменная, предваряющая скобки, задаёт количество итераций
Добавим аккумулятор number_of_iterations для хранения числа, хранящегося в аккумуляторе var_acc и аккумулятор index_acc
для индексной переменной i
Если в тексте программы встретилась открывающая скобка [ то запоминаем её положение в индексной переменной,
а переменнцю var_acc загружаем в number_of_iterations, сбрасываем флаг 

 if str[i]='[' then
 begin
   index_acc := i;
   number_of_iterations := var_acc;
   flag := false;
 end;
Если в тексте программы встретилась закрывающая скобка ] и если переменная number_of_iterations
больще нуля, то декркментируем переменную number_of_iterations и переходим к открывающей скобке 
 if str[i]=']' then
 begin
   arr[number_of_iteration-97] := arr[number_of_iteration-97]-1; 
   i := index_acc;
 end;
Если в тексте программы встретилась обратная кавычка, переходим к следующему символу в строке str[]
и выводим на экран требуемый элемент массива arr[]
https://github.com/demsp/pas/blob/master/abcd/iterations.pas /p>

Для разбора выражения, содержащего '+','-','*','/', добавим добавим арифетический опрератор arithmetic_operator
и аккумулятор для арифметического оператора operator_acc
Пускай арифетическое выражение может содержать лишь четыре арифетических опретора 

   ::=  { }
   ::= '+' | '-' | '*' | '/'
При разборе выражения необходимо учитывать приоритет операторов.
