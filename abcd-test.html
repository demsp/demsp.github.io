Пусть цикл обрабатывается внутри квадратных скобок.
Пусть переменная, предваряющая скобки, задаёт количество итераций.
Аккумулятор number_of_iterations служит для хранения числа, содержащегося в аккумуляторе var_acc
аккумулятор index_acc нужен для индексной переменной i
Если в тексте программы встретилась открывающая скобка [ то её положение сохраняется в индексной переменной, а переменная var_acc загружается в number_of_iterations
флаг сбрасывается 

 if str[i]='[' then
 begin
   index_acc := i;
   number_of_iterations := var_acc;
   flag := false;
 end;
Если в тексте программы встретилась закрывающая скобка ] и если переменная number_of_iterations
больше нуля, переменная number_of_iterations декрементируется и происходит переход к открывающей скобке 
 if str[i]=']' then
 begin
   arr[number_of_iteration-97] := arr[number_of_iteration-97]-1; 
   i := index_acc;
 end;
Если в программе str[] встретилась обратная кавычка, происходит переход к следующему символу в строке str[]
и выводится на экран требуемый элемент массива arr[]
следовательно точку с запятоий после выводимой переменной ставить не обязательно https://github.com/demsp/pas/blob/master/abcd/iterations.pas
Пусть переменные a и b хранят соседние числа Фибоначчи (инициализируются единицами), в переменной с вычисляется сумма этих чисел, переменная d выступает в качестве счетчика итераций. Тогда программа, выводящая на экран ряд Фибоначчи выглядит так: str:= ' a=1; b=1; d=9; [ c=a+b; a=b; b=c; 'b; ] ';

Пускай в массиве хранятся вещественные числа
arr: array[0..4] of real;
Добавим оператор деление /
Пусть переменная bin_operator содержит текущий арифметический оператор
данный символ загружается в переменную bin_operator, если в программе str[] встретились знаки + или /
if str[i]='+' then bin_operator:='+';
if str[i]='-' then bin_operator:='-';
данное число прибавляется к аккумулятору,если в программе str[] встретилось числовое значение и текущим оператором (т.е. символом, содержащимся в переменной bin_operator) является +
if bin_operator='+' then acc:=acc+(Ord(str[i])-48);
число из аккумулятора делится на данное обрабатываемое число и результат загружается в acc,если текущим оператором является /

Пускай арифетическое выражение может содержать лишь четыре арифетических опретора
<Expr> ::= <Number> {<Operation> <Number>}
<Operation> ::= '+' | '*'
Фигурные скобки (скобки Клини) обозначают, что к числу рекурсивно присоединяется пара: оператор и число
и таким образом к.п.г. реализуется цепочка, состоящая из разделённых операторами чисел.
Если все операторы имеют одинаковый приоритет, то вычисление результата можно выполнить непосредственно в процессе разбора арифетического выражения, но в данном случае приоритет оператора * выше приретета +
